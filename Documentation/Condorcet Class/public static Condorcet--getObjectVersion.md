## public static Condorcet::getObjectVersion

### Description    

```php
public static Condorcet::getObjectVersion ( [ bool type = false] )
```

Get the library version.    


##### **type:** *bool*   
* true will return : '2.0'
* false will return : '2.0.0'    



### Return value:   

*(string)* Condorcet PHP version.
