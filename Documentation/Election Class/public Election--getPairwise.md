## public Election::getPairwise

### Description    

```php
public $Election -> getPairwise ( [ bool explicit = true] )
```

Return the Pairwise.    


##### **explicit:** *bool*   
Get an explicit array with candidates names as keys.    



### Return value:   

Pairwise object. Or an explicit array if _explicit_ argument == true.
