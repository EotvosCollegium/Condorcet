## public Vote::removeAllTags

### Description    

```php
public $Vote -> removeAllTags ( mixed tags )
```

Remove all registered tag(s) on this Vote.    


##### **tags:** *mixed*   
They can be added by string separated by commas or an array.    



### Return value:   

*(bool)* Return True.
