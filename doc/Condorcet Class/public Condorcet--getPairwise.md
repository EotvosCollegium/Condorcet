## public Condorcet::getPairwise

### Description    

```php
public $Condorcet -> getPairwise ( [ bool explicit = true] )
```

Return the Pairwise.    


##### **explicit:** *bool*   
Get an explicit array with candidates names as keys.    



### Return value:   

Pairwise object. Or an explicit array if _explicit_ argument == true.
