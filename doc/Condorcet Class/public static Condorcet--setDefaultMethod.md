## public static Condorcet::setDefaultMethod

### Description    

```php
public static Condorcet::setDefaultMethod ( string method )
```

Put a new static method by default for the news Condorcet objects.    


##### **method:** *string*   
Method Name.    



### Return value:   

(bool) In case of success, return TRUE
