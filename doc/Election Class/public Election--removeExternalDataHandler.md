## public Election::removeExternalDataHandler

### Description    

```php
public $Election -> removeExternalDataHandler ( )
```

Remove an external driver to store vote on very large election. And import his data into classical memory.    


### Return value:   

*(bool)* True if success. Else throw an Exception.
